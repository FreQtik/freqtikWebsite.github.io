FILES ARE PATCHED FROM YOUR CURRENT REPOSITORY
==============================================

This package intentionally does not contain a hand-reconstructed index.html or a guessed copy of your large JavaScript file.

Instead, it performs narrow edits against the exact current files already in your local GitHub repository:

- impulse-anvil.html is cleaned and copied byte-for-byte to index.html after canonical URL changes.
- connect.html is edited only in its metadata, hero and the two requested card paragraphs.
- assets/freqtik-site.js receives matching fallback edits.
- the old process-focused AI wording is removed from public root HTML and machine-readable summaries.
- all originals are backed up before writing.

This is safer than replacing the site with a reconstructed package because it preserves your current product sections, audio players, images, links, IDs, scripts and any changes made since the previous package.
